# Keylogger-Detector-for-Android-Beginner-Project-
Project Name: Android Keylogger Detector (Simulation)
Description:
This project simulates how cybersecurity tools detect keyloggers.
It checks installed apps and looks for suspicious keywords like spy, monitor, logger, etc.
If any app name contains those terms, it warns the user.
Purpose:
To understand the basics of malware detection by scanning app names and identifying potential threats.
